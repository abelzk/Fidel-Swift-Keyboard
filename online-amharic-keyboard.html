How to Type

Fidel Swift uses a consonant + vowel key system. Type a consonant first, then press a vowel key to change its order. Standalone vowel keys like a, e, i, o, and u are vowel modifiers, not independent letters. To type independent vowels, use the x family.

Vowel Order

base → 1st order
u → 2nd order
i → 3rd order
a → 4th order
y → 5th order
e → 6th order
o → 7th order
] → 8th order
