How to Type

Fidel Swift uses a consonant + vowel key system. Each consonant has 8 vowel orders (forms). Type a consonant, then a vowel key to change its order:

base → h → ሀ 1st
u → hu → ሁ 2nd
i → hi → ሂ 3rd
a → ha → ሃ 4th
y → hy → ሄ 5th
e → he → ህ 6th
o → ho → ሆ 7th
] → h] → ኋ 8th
Independent vowels — use x instead of a consonant: xa → ኣ xi → ኢ xo → ኦ x] → ኧ

About the Amharic Language & Ge'ez Script (ፊደል)

Amharic (አማርኛ) is the official working language of Ethiopia and is spoken by over 35 million native speakers and 25 million second-language speakers worldwide — including large Ethiopian diaspora communities in the United States, Israel, Germany, the United Kingdom, and the Middle East. It is the second most spoken Semitic language after Arabic.

Amharic is written in the Ge'ez script (also called Fidel, ፊደል), one of the world's oldest writing systems still in active use — over 2,000 years old. It is an abugida (syllabary) where each character represents a consonant-vowel combination. Amharic has 33 base consonants, each with 7 vowel forms, producing over 230 unique characters. The script is written left-to-right and is also used for Tigrinya, Tigre, and other Ethiopic languages.

Most standard keyboards do not include Amharic characters, making online tools like Fidel Swift essential for Ethiopians abroad, students of the language, and anyone who needs to write in Amharic without reconfiguring their system settings.

How do I type Amharic online without installing software?

Simply open this page in any browser and start typing with your regular keyboard. Fidel Swift converts your English keystrokes into Amharic Ge'ez characters instantly using phonetic transliteration — no download, plugin, or system setting change required.

Does this Amharic keyboard work on iPhone and Android?

Yes. Fidel Swift's online Amharic keyboard is fully responsive and works on all modern mobile browsers, including Safari on iPhone and Chrome on Android. You can type, copy, and paste Amharic text into WhatsApp, Telegram, social media, or any app.

What is the difference between phonetic and layout-based Amharic keyboards?

A layout-based keyboard maps Amharic characters to specific physical keys — requiring memorization. Fidel Swift uses phonetic transliteration: you type the sound of the Amharic character using English letters (e.g., s l a m e → ሰላም), making it intuitive for anyone who knows how the language sounds.

Can I use this keyboard to type Tigrinya?

Many Ge'ez script characters are shared between Amharic and Tigrinya. While Fidel Swift is optimized for Amharic, the core Fidel characters it supports cover a large portion of Tigrinya text as well.

Is my typed Amharic text saved or stored?

No. All typing happens locally in your browser. Your text is never sent to any server or stored online, ensuring complete privacy.
